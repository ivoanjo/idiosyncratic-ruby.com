Ruby Lookalikes

Tired of good old, but slow Ruby? Try one of Ruby's successors:

Name Compiles To Description
Crystal Machine Fast Ruby
Mirah JVM Fast Ruby in Java
Opal JS Ruby for JavaScript
CoffeeScript JS JavaScript for Rubyists
Elixir BEAM Ruby for Erlang
Scala JVM Multi-Paradigm Ruby
Rooby Machine Simple Ruby
Rubex Machine Ruby Extensions Ruby
Goby Machine Go meets Ruby

All of them are better than Ruby, in some way!
